## Format
- first line header
- column
	- date: 2017-01-01
	- min: 일일 최저기온
	- average: 일일 평균기온
	- max: 일일 최고기온

## Meta
- region: 대구, 서울, 광주, 강릉, 인천, 제주
- period: 1960-01-01 ~ 2017-12-31
- null value linear interpolation
